# LedgerLens AI

LedgerLens AI is a professional Receipt & Invoice Digitizer built using
Streamlit and Google Gemini Flash.

## Features
- Receipt & invoice upload
- Image preprocessing
- AI-based JSON extraction
- Persistent SQLite storage
- Spending analytics dashboard
- Secure API key handling
- Voice confirmation (Text-to-Speech)

## Run Locally
pip install -r requirements.txt
streamlit run app.py

## Deployment
Fully compatible with Streamlit Cloud.
